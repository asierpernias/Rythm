# Rythmy ![](https://hackatime.hackclub.com/api/v1/badge/U09F98XRE1G/asierpernias/Rythm)
a one key web rythm game

## How to play?
- While in the menu: use space to select the difficultie
- While playing: Use space when the note is near enough the hit zone / keep the space if it is a holded note
- Game over: Select a 6 characters name tag to keep your puntuation on the leaderboard (you will be able to use it again if you give it another try)

### Puntuation

| Golpe | Puntos base |
|-------|-------------|
| PERFECT | 100 |
| GOOD | 50 |
| PERFECT HOLD | 150 |


Every 10 combos you'll get a multiplier (max x4)

## Features 
- 3 second countdown
- Dramatic shake when loosing a life
- Normal/Holded notes
- Leadeboard with the top 5
- Beat clock (integrated with the bpm)
- Particles and animations with js

## Firebase
Apart from HTML/CSS/JS I used a Firebase Realtime Database to display the leaderboard of the top 5 puntuations.
